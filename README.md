# Expense Tracker (Трекер расходов)

##  Автор
Сунгатуллин Амир

##  Описание программы
Простое GUI-приложение на Python (tkinter) для учёта личных финансов. Позволяет добавлять расходы, фильтровать их по категории и периоду, автоматически подсчитывать сумму за выбранный промежуток времени, а также сохранять и загружать данные в формате JSON.

##  Технологии
- Python 3.x
- tkinter / ttk (стандартная библиотека GUI)
- json, datetime, os

## Как использовать
1. Убедитесь, что установлен Python 3.6+.
2. Запустите приложение: `python expense_tracker.py`
3. Введите сумму, выберите категорию и укажите дату (формат `ГГГГ-ММ-ДД`).
4. Нажмите **"Добавить"** → запись появится в таблице и автоматически сохранится.
5. Используйте поля фильтра и кнопку **"Применить"** для расчёта суммы за период.
6. Кнопки **"Сохранить JSON"** / **"Загрузить JSON"** позволяют вручную работать с файлом `expenses.json`.

##  Примеры использования (Тест-кейсы)
| Действие | Ожидаемый результат |
|----------|---------------------|
| Ввод суммы `-50` | Ошибка: "Сумма должна быть положительным числом." |
| Ввод даты `2024-32-01` | Ошибка: "Неверный формат данных." |
| Фильтр "Категория: Еда", период `2024-05-01` → `2024-05-31` | В таблице только расходы за май в категории Еда, внизу подсчитана сумма |
| Нажатие "Сбросить" после фильтра | Возвращается полный список, сумма сбрасывается на общую |
| Закрытие и повторный запуск | Данные автоматически подгружаются из `expenses.json` |
